Project 2:Guess the Number Game - Android App

- Developed during: 2-2 semester

Guess game is an mobile application with built in random numbers from 1 to 50 ,Users have to guess the correct number to win he game.

    Languages & Technologies Used:

  • Java: Core programming language for Android app development. Used for logic, UI handling, and event-driven programming.
  • XML: Used for designing the graphical user interface (GUI). Defines layouts, buttons, text views, and other UI elements.
  • Android SDK: Provides libraries and tools to develop, debug, and test the app on various Android devices.
  • Gradle: Used for project build automation and APK generation

    Skills Acquired:

  • Android App Development: Understanding of activity lifecycle, event handling, and UI components.
  • Object-Oriented Programming (OOP): Implemented through Java, covering concepts like classes, methods, and event listeners.
  • User Interface (UI) Design: Created a responsive layout using XML and ConstraintLayout for better device compatibility.
  • Random Number Generation & Logic Handling: Implemented number guessing logic using Java’s Random class.
  • APK Generation: Built the app package file (.apk) using Android Studio for deployment on Android devices.
